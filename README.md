# FastAPI ML Deployment (Learning Project) 🚀

A personal sandbox project created to learn the fundamentals of transitioning a Machine Learning model into a REST API.

This repository demonstrates how to serve a pre-trained `scikit-learn` model using modern Python tools and strict data validation.

## 🛠️ Tech Stack

- **Framework:** FastAPI & Uvicorn
- **Data Validation:** Pydantic (V2)
- **Machine Learning:** scikit-learn, Pandas
- **Environment:** `uv`
